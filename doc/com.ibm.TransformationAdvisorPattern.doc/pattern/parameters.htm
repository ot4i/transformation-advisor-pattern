Parameters for the Transformation Advisor pattern

Analysis Options
Pattern Parameter Default Description
BAR File

The BAR File parameter specifies which BAR file should be used by the Transformation Advisor when searching for message flows to be analysed.
ACE software Selected

Selecting this option indicates that the Transformation Advisor should check aspects of compatability for deployment to App Connect Enterprise software.
ACE on IBM Cloud Private Selected

Selecting this option indicates that the Transformation Advisor should check aspects of compatability for deployment to App Connect Enterprise on IBM Cloud Private.
ACE on IBM Cloud (public) Selected

Selecting this option indicates that the Transformation Advisor should check aspects of compatability for deployment to the App Connect Enterprise on Cloud service (a managed service part of the public IBM Cloud).

All the pattern parameters are listed with their default values. Where no default is indicated, the default value is an empty string.
